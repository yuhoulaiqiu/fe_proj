# 社区便民公益互助平台（React + Go）

面向社区居民/志愿者/物业工作人员的演示项目，提供用户端社区服务入口 + 管理端后台能力。主要包含三大业务模块：公益活动、便民服务目录、失物招领；其中“失物招领”在管理端实现完整 CRUD（增删改查）。

## 功能点汇总

### 用户端
- 账号体系：注册、登录、获取当前用户信息（token 会话）
- 公益活动
  - 活动列表/详情（分页）
  - 发布活动、编辑/删除活动（仅发布者）
  - 报名/取消报名（报名截止：开场前 24 小时）
  - 查看报名名单（仅发布者）与导出报名 CSV
  - 我发布/我报名列表
  - 活动状态自动演进（active/closed/finished/cancelled）
- 通知中心
  - 报名后生成提醒（开场前 24h、1h），支持已读标记
  - 后台定时任务补齐/修复提醒数据
- 便民服务目录
  - 服务列表/详情（分页）
- 失物招领
  - 记录列表/详情（只读浏览）

### 管理端
- 管理员鉴权（admin role）
- 失物招领后台管理
  - 列表查询
  - 新增/编辑
  - 删除（软删除）
